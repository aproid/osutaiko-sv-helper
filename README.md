# osu!taiko SV Helper
Simple utility for adjusting slider velocity for osu!taiko mapping

## Requirement
* Node.js 14.16.0 or later

## Build
#### Serve Application For Development
```
$ cd PROJECT_FOLDER
$ npm install
$ npm start
```
#### Build Executable File
```
$ cd PROJECT_FOLDER
$ npm install
$ npm build:win64  ---  64bit
$ npm build:win32  ---  32bit
```
